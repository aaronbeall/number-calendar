# Numbers Go Up
A simple calendar for showing numbers (P&amp;L, etc)

## Features

* Calendar grid view with previous, next, and today navigation
* Each day can be edit and display numbers. Numbers are entered like "1+2-5" and parsed into series data like `[1, 2, -5]`
* Display weekly and monthly total, median, mean, min, max
* Numbers are displayed with positive (green) and negative (red) styling
* Below the calendar is a bar graph of the data, with option to toggle chart as Serial or Cumulative
* Modern, polished, simple and elegant design principles
